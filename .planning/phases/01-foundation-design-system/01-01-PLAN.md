<objective>
Scaffold the entire monorepo from scratch: `/frontend` (Next.js 15 App Router), `/backend` (Express + TypeScript), and `/admin` (React + Vite). Wire all three together with a root-level `npm run dev` via `concurrently` so all three start with one command. Create `tailwind.shared.config.ts` at the repo root with RSG's color tokens — imported by both `frontend` and `admin` Tailwind configs. Result: `npm run dev` starts all three services and each returns a 200 response on its port.
</objective>
